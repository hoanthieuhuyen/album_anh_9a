Hướng Dẫn Lưu Tệp Offline

Bước 1: Lưu mã nguồn về máy

Sao chép toàn bộ mã nguồn HTML về máy tính của bạn.

Bước 2: Thư viện PageFlip Offline

  1. Tải file JS: page-flip.browser.min.js
  2. Đặt cùng thư mục với file HTML.
  3. Sửa thẻ script thành: <script src="./page-flip.browser.min.js"></script>

Bước 3: Nhạc nền Offline

Tải file MP3 về, đặt cùng thư mục, sửa thẻ <source src="..."> thành đường dẫn nội bộ.

Cách Đưa Ảnh Lớp Bạn Vào

1. Nhúng Ảnh Từ Google Drive

  1. Chia sẻ ảnh Google Drive ở chế độ "Bất kỳ ai có link đều xem được".
  2. Lấy ID ảnh từ link (chuỗi sau /d/).
  3. Dùng cấu trúc: <img src="https://drive.google.com/thumbnail?id=ID&sz=w1000">

2. Thay đổi Lời tự sự và Chữ ký

  • Sửa trực tiếp tên thành viên và nội dung trong thẻ có class diary-text.
  • Chữ ký thật: chụp ảnh, xóa nền tại remove.bg, upload Drive rồi thay thẻ img.
