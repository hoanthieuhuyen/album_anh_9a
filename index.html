Hướng Dẫn Lưu Tệp Lên Máy Tính Chạy Offline

Bước 1: Lưu mã nguồn về máy

Bên cạnh giao diện xem trước này, bạn hãy nhìn sang góc phải màn hình, sao chép toàn bộ mã nguồn HTML hoặc nhấn nút tải trực tiếp file "album_lat_trang_tuong_tac.html" về máy tính của bạn.

Bước 2: Chuẩn bị thư viện chạy Offline 100%

Để cuốn sách lật có hiệu ứng 3D mượt mà ngay cả khi rút dây mạng / mất wifi hoàn toàn, bạn cần tải thư viện điều khiển về lưu cùng thư mục:

  1. Truy cập đường dẫn: Tải file JS này.
  2. Nhấp chuột phải chọn Lưu dưới dạng... (Save as...) và đặt tên là page-flip.browser.min.js.
  3. Đặt file này nằm **chung một thư mục** với file album_lat_trang_tuong_tac.html bạn vừa tải về.
  4. Mở file album_lat_trang_tuong_tac.html bằng Notepad hoặc trình biên dịch code, kéo xuống gần cuối trang tìm thẻ:
    <script src="https://cdn.jsdelivr.net/npm/page-flip@2.0.7/dist/js/page-flip.browser.min.js"></script> Thay đổi đường dẫn thành đường dẫn nội bộ (local) như sau:
    <script src="./page-flip.browser.min.js"></script>

Bước 3: Nhạc nền Offline

Tương tự như nhạc nền, bạn hãy tải về một bài nhạc MP3 yêu thích (Ví dụ: mong_uoc_ky_niem_xua.mp3), đặt chung thư mục, rồi thay đổi thẻ <source src="..."> ở phía cuối file thành:
<source src="./mong_uoc_ky_niem_xua.mp3" type="audio/mpeg">

Cách Đưa Ảnh & Bút Tích Thực Của Lớp Bạn Vào

1. Cách Nhúng Ảnh Từ Google Drive (Chuẩn Xác Nhất)

Để đưa ảnh của lớp hoặc của thành viên đã lưu trên Google Drive vào album điện tử này, hãy làm theo các bước chuẩn xác sau:

  1. Nhấp chuột phải vào file ảnh trên Google Drive của bạn, chọn Chia sẻ (Share) -> Bất kỳ ai có đường liên kết này đều xem được (Anyone with link can view).
  2. Sao chép đường liên kết chia sẻ đó. Đường link sẽ có cấu trúc dạng:
    https://drive.google.com/file/d/1JTKzKCvieoom5jYcrReA2RAknCx-BoSh/view?usp=sharing
  3. Sao chép phần ID của ảnh (chuỗi ký tự nằm sau phần /d/ và trước /view).
  4. Dán ID ảnh đè lên chuỗi chữ DIEN_ID_ANH_... trong thuộc tính src với cấu trúc link **Thumbnail** ổn định này: <img src="https://drive.google.com/thumbnail?id=ID_CUA_BAN&sz=w1000" ...>

2. Thay đổi Lời tự sự và Chữ ký

  • Sửa nội dung văn bản: Sửa đổi trực tiếp tên thành viên, chức danh lớp và những dòng tự sự tràn đầy hoài niệm nằm trong cặp thẻ có Class diary-text.
  • Xử lý chữ ký tay thật: Bạn có thể bảo mọi người dùng bút mực đậm màu (đen hoặc xanh nước) viết chữ ký lên một tờ giấy trắng phẳng sạch, dùng điện thoại chụp thẳng từ trên xuống. Đưa bức ảnh đó lên trang web remove.bg để lọc bỏ hoàn toàn nền trắng (chỉ giữ lại nét mực trong suốt định dạng `.png`), sau đó tải ảnh lên Google Drive và thay đổi bằng thẻ chèn ảnh đè lên phần chữ mẫu: <img src="https://drive.google.com/thumbnail?id=ID_CHU_KY_TU_DRIVE&sz=w1000" style="height: 60px;" alt="Chữ ký">
